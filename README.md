Strivio is a safety-first route discovery app designed for runners and walkers. It helps users find better, safer paths based on location, time of day, and community-reported conditions.

---

## Overview

Strivio focuses on solving a simple but important problem:
**finding safe and reliable routes for outdoor movement.**

Unlike traditional navigation apps like Waze that prioritize traffic and driving efficiency, Strivio is built specifically for **pedestrians, runners, and fitness-focused users**.

---

## MVP Features

* **Route Mapping**

  * Visualize curated jogging and walking routes on an interactive map

* **Safety Scoring**

  * Routes are labeled as **Good / Moderate / Avoid**
  * Based on time-of-day logic and reported hazards

* **Hazard Reporting**

  * Users can report:

    * Broken sidewalks
    * Poor lighting
    * Unsafe areas

* **Route Insights**

  * Distance
  * Difficulty
  * Suggested best time to use

---

## How It Works

Strivio uses a combination of:

* predefined route data
* time-based safety assumptions
* community-generated hazard reports

to generate a simple, understandable safety score for each route.
